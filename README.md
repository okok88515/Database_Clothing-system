# 服飾店管理系統

## 📝 專案簡介
這是一個為實體服飾店打造的庫存管理系統，支援店家、供應商、存貨等資訊的新增與查詢。

## 🧰 技術使用
- Java
- MySQL
- jQuery

## 🔧 功能簡述
- 新增、編輯、刪除存貨
- 管理供應商資訊
- 前端與資料庫串接

## 🚀 執行方式
1. 匯入資料庫
2. 執行主程式
3. 於本機 localhost 操作系統
